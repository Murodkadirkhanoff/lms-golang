# LMS Project

## Project Goal

Learning Management System.

## Stack

Frontend:
- Next.js 16
- TypeScript
- Tailwind
- shadcn/ui
- TanStack Query

Backend:
- Go
- Chi Router
- PostgreSQL
- JWT

## Architecture

Feature-based architecture.

src/
├── app
├── features
├── components
├── services
├── hooks

## Design Rules

- Mobile First
- Clean SaaS Design
- Minimalistic
- Professional UI

## Coding Rules

- No any types
- Use TypeScript interfaces
- Use React Query
- Use Server Components when possible
